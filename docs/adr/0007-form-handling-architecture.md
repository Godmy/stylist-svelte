# ADR 0007: Архитектура form-handling через доменные контракты

**Status:** Accepted (Revised)
**Date:** 2025-01-04
**Revised:** 2026-02-20

## Контекст
Формы используются в десятках компонентов, но отдельный централизованный `src/lib/types/form.ts` отсутствует.

## Решение
Form-контракты и поведенческие соглашения ведём в доменных модулях design-system:
- `props/interaction-forms.ts`, `props/input.ts`, `props/common.ts`;
- соответствующие `styles/*`, `state/*`, `models/*` для runtime-логики.

## Проверка по кодовой базе
- `src/lib/types/form.ts` отсутствует.
- Форменные контракты и стилизация реально распределены по design-system слоям.

## Последствия
- Логика форм эволюционирует рядом с реальными компонентами.
- Упрощается масштабирование сложных форменных сценариев.

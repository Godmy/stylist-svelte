# ADR 0006: Co-located stories рядом с компонентами

**Status:** Accepted (Revised)
**Date:** 2025-11-04
**Revised:** 2026-02-20

## Контекст
Истории должны развиваться вместе с компонентами, иначе они быстро рассинхронизируются.

## Решение
Храним `*.story.svelte` рядом с реализацией компонентов в `src/lib/components/**`.

## Проверка по кодовой базе
- Co-located stories присутствуют во всех основных доменах компонентов.
- Playground-совместимость и story-точка входа поддерживаются в проекте.

## Последствия
- Снижается риск устаревших примеров.
- Ревью охватывает компонент и его документацию в одном изменении.

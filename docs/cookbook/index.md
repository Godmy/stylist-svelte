# Cookbook — книга рецептов

Каждое замечание скрипта аудита ссылается на рецепт из секции `ERROR/` или `WARNING/`.  
Секция `SVELTE/` содержит паттерны написания компонентов на Svelte 5.

---

## ERROR

| Код | Название | Описание |
|-----|----------|----------|
| [ERROR/001](./ERROR/001/) | Нет экспортов | Файл `index.ts` не содержит ни одного `export` |
| [ERROR/002](./ERROR/002/) | Множественные экспорты | Файл содержит более одного `export` |
| [ERROR/003](./ERROR/003/) | Реэкспорт запрещён | Найден `export { }` или `export * from` вне баррельного контекста |
| [ERROR/004](./ERROR/004/) | Неверный тип экспорта | Тип экспорта не совпадает с кластером |
| [ERROR/005](./ERROR/005/) | Отсутствует export default | Файл `*.svelte.ts` не имеет `export default` |

## WARNING

| Код | Название | Описание |
|-----|----------|----------|
| [WARNING/001](./WARNING/001/) | Неверное имя файла | TypeScript-файл назван не `index.ts` |
| [WARNING/002](./WARNING/002/) | Legacy-структура | `index.ts` находится рядом с другими `.ts`-файлами |
| [WARNING/003](./WARNING/003/) | Неэкспортируемое объявление | Объявление верхнего уровня не имеет `export` |

## SVELTE

| Код | Тема |
|-----|------|
| [SVELTE/001](./SVELTE/001/) | `children` / `Snippet` вместо `<slot>` |
| [SVELTE/002](./SVELTE/002/) | Миграция обработчиков событий (`on:click` → `onclick`) |
| [SVELTE/003](./SVELTE/003/) | TypeScript-паттерны (`$props`, `HTMLAttributes`, индексация) |
| [SVELTE/004](./SVELTE/004/) | Доступность (a11y): label, role, keyboard, nesting |

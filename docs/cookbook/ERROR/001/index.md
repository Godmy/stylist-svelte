# ERROR-001: Нет экспортов

**Серьёзность:** ❌ Error  
**Правило ADR:** Файл в кластерной папке обязан экспортировать ровно одну сущность.

---

## Что произошло

Скрипт нашёл файл `index.ts`, который не содержит ни одного `export`.  
Это может быть:
- файл-заглушка, которую забыли заполнить
- служебный файл (README, заметки), оказавшийся внутри кластерной папки
- баррель, у которого нет ни одной дочерней папки с содержимым

---

## Примеры

```ts
// ❌ Пустой файл
```

```ts
// ❌ Есть код, но нет export
const helper = () => {};
```

```ts
// ❌ README.md оказался в папке семейства
// color/const/record/colors-scales/README.md
```

---

## Как исправить

### Случай 1 — Файл-заглушка

Если семейства ещё нет, удалить папку целиком. Баррель выше пересоберётся автоматически.

```
# Удалить:
calendar/type/record/    ← если пустая или без содержимого
```

### Случай 2 — Служебный файл в кластерной папке

Удалить. Документация не должна лежать внутри кластерной папки.  
Держите `README.md` выше — на уровне домена или кластера.

### Случай 3 — Забытый export

Добавить ключевое слово `export` перед объявлением:

```ts
// ✅ Было: const X = ...
// Стало:
export const X = ...
```

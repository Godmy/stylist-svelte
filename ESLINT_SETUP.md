# Настройка ESLint в проекте stylist-svelte

## Обзор

Проект stylist-svelte использует @kitql/eslint-config для унификации стиля кода и обеспечения качества кода.

## Установка

```bash
yarn add -D @kitql/eslint-config
```

## Конфигурация

Файл конфигурации ESLint расположен в `.eslint.config.js`:

```js
import { kitql } from '@kitql/eslint-config';

export default [...kitql()];
```

## Использование

### Проверка кода

Для проверки кода выполните:

```bash
yarn lint
```

### Форматирование кода

Для автоматического форматирования кода выполните:

```bash
yarn format
```

## Особенности

- Конфигурация основана на современных стандартах JavaScript/TypeScript
- Включает правила для Svelte компонентов
- Обеспечивает согласованность кода во всем проекте
- Интегрируется с Prettier для форматирования

## Интеграция с IDE

Для лучшей интеграции с вашей IDE убедитесь, что установлены соответствующие плагины:

- ESLint плагин
- Prettier плагин
- Svelte плагин

## Обновление конфигурации

При обновлении @kitql/eslint-config может потребоваться обновить конфигурацию проекта. Проверяйте изменения в документации пакета.

# Атомы (Atoms)

Атомы - это базовые строительные блоки пользовательского интерфейса. Это самые простые компоненты, которые нельзя разделить на более мелкие части. Они представляют собой одиночные элементы интерфейса, такие как кнопки, поля ввода, иконки и т.п.

## Структура

Компоненты атомов организованы по функциональным категориям:

### [Layout (Компоновка)](#layout-components)
Компоненты для структурирования и распределения элементов интерфейса.

### [Typography (Типографика)](#typography-components)
Компоненты для отображения текстового контента и стилизации текста.

### [Controls (Элементы управления)](#controls-components)
Интерактивные элементы для взаимодействия пользователя с интерфейсом.

### [Input (Поля ввода)](#input-components)
Компоненты для ввода данных пользователем.

### [Feedback (Обратная связь)](#feedback-components)
Компоненты для предоставления информации пользователю о состоянии системы.

### [Media (Медиа и иконки)](#media-components)
Компоненты для отображения изображений, иконок и других медиаэлементов.

### [Placeholders (Заглушки)](#placeholders-components)
Компоненты-заглушки для показа во время загрузки контента.

### [Process (Процесс)](#process-components)
Индикаторы процессов, такие как прогресс-бары и спиннеры.

### [Chat (Чат)](#chat-components)
Компоненты, специфичные для чат-приложений и коммуникации.

---

## Layout Components

Компоненты для структурирования и распределения элементов интерфейса.

### Divider
Горизонтальная или вертикальная линия для разделения контента.

### Container
Контейнер с ограниченной шириной и центрированием содержимого.

### Stack
Компонент для равномерного распределения пространства между элементами.

### Spacer
Пустой элемент для добавления пространства между компонентами.

### AspectRatio
Компонент для сохранения соотношения сторон у содержимого.

---

## Typography Components

Компоненты для отображения текстового контента и стилизации текста.

### Text
Основной компонент для отображения текста с различными вариантами стилизации.

### Heading
Компонент для заголовков с различными уровнями.

### Link
Компонент для создания гиперссылок.

### Badge
Маленький элемент для отображения статуса или подсчета.

### Chip
Компонент для отображения небольших информационных блоков.

### Tag
Тег для маркировки или категоризации элементов.

### CodeBlock
Блок для отображения форматированного кода.

### InlineCode
Компонент для отображения кода внутри текста.

### Kbd
Компонент для отображения сочетаний клавиш.

### Label
Текстовая метка для элементов формы.

### Abbr
Компонент для аббревиатур с возможностью отображения пояснения.

### Blockquote
Компонент для цитат с соответствующим оформлением.

### Em
Компонент для курсивного акцентного текста.

### Strong
Компонент для жирного акцентного текста.

### Small
Компонент для маленьких вспомогательных текстов.

### Strikethrough
Компонент для зачеркнутого текста.

### Superscript и Subscript
Компоненты для надстрочного и подстрочного текста.

### DefinitionTerm и DefinitionDescription
Компоненты для терминов и описаний в определениях.

### Highlight
Компонент для выделения важного текста.

---

## Controls Components

Интерактивные элементы для взаимодействия пользователя с интерфейсом.

### Button
Универсальная кнопка с различными стилями и состояниями.

### Input
Текстовое поле для ввода данных с различными состояниями и вариантами.

### Textarea
Многострочное поле ввода текста.

### Select
Выпадающий список для выбора одного из нескольких вариантов.

### Checkbox
Флажок для выбора одного или нескольких вариантов.

### Radio
Радиокнопка для выбора одного из нескольких вариантов.

### Toggle
Переключатель для включения/отключения определенного состояния.

### Switch
Более крупный переключатель с визуальным представлением состояния.

### RangeSlider
Слайдер для выбора значения в заданном диапазоне.

### InputAddon
Префикс или суффикс для поля ввода (иконка, текст).

### ColorSwatch
Образец цвета (квадрат).

### SliderTick
Одна отметка на шкале RangeSlider.

### CloseButton
Стандартизированная кнопка "закрыть" (крестик).

---

## Input Components

Компоненты для ввода данных пользователем.

### FormHelperText
Текст-подсказка под полем ввода.

### FormErrorMessage
Сообщение об ошибке для поля.

### PinInputDigit
Одна ячейка (цифра) для ввода пин-кода.

### CharacterCount
Индикатор текущее/макс для полей ввода.

---

## Feedback Components

Компоненты для предоставления информации пользователю о состоянии системы.

### Alert
Компонент для отображения важных сообщений.

### Dot
Маленький круглый индикатор для подсветки активности или уведомлений.

### Counter
Компонент для отображения числового счетчика.

### ProgressCircle
Круговой индикатор прогресса.

---

## Media Components

Компоненты для отображения изображений, иконок и других медиаэлементов.

### Icon
Отображение иконки из коллекции иконок.

### Avatar
Отображение пользовательского аватара с поддержкой статусов.

### Image
Улучшенный компонент изображения с поддержкой ленивой загрузки и заглушек.

### Favicon
Компонент для отображения иконки сайта по URL.

### CountryFlag
Компонент для отображения флага страны по ее коду.

### IconWrapper
Контейнер для иконки, задающий ей фон, форму и размеры.

---

## Placeholders Components

Компоненты-заглушки для показа во время загрузки контента.

### Skeleton
Заглушка с анимацией для загрузки данных.

### TableSkeleton
Заглушка для таблицы во время загрузки данных.

### SkeletonText
Заглушка в виде строки текста.

### SkeletonCircle
Круглая заглушка (для аватаров, иконок).

### SkeletonRectangle
Прямоугольная заглушка (для изображений, карточек).

## Process Components

Индикаторы процессов, такие как прогресс-бары и спиннеры.

### Spinner
Анимированный индикатор загрузки.

### ProgressBar
Визуальное представление прогресса выполнения задачи.

---

## Chat Components

Компоненты, специфичные для чат-приложений и коммуникации.

### ChatStatusIndicator
Индикатор статуса участника чата (онлайн, оффлайн, печатает).

### MessageStatus
Индикатор статуса сообщения (отправлено, доставлено, прочитано).

### MessageTimestamp
Отображение времени отправки сообщения.

## Использование

Каждый атом можно импортировать отдельно:

```svelte
<script>
  import { Button, Input, Alert } from 'stylist-svelte';
</script>

<Button variant="primary">Кнопка</Button>
<Input label="Имя" placeholder="Введите имя" />
<Alert variant="info">Информационное сообщение</Alert>
```

Атомы могут быть использованы как основа для построения более сложных компонентов (молекул и организмов), что обеспечивает единообразие и последовательность в интерфейсе.

## Принципы

- **Простота**: Каждый атом выполняет одну конкретную функцию
- **Повторное использование**: Атомы могут быть использованы в различных контекстах
- **Согласованность**: Атомы обеспечивают единообразный стиль и поведение
- **Гибкость**: Атомы могут быть настроены через пропсы
- **Доступность**: Все атомы обеспечивают доступность в соответствии с WCAG


# Компоненты библиотеки stylist-svelte

Этот документ описывает все компоненты в проекте `stylist-svelte`, организованные по архитектуре Atomic Design: атомы, молекулы и организмы.

## Атомы

Атомы - это базовые строительные блоки пользовательского интерфейса, которые не могут быть разбиты на более мелкие компоненты.

### Список атомов

| Компонент | Описание |
|----------|----------|
| `Alert` | Компонент для отображения важных сообщений пользователю с различными вариантами (информация, успех, предупреждение, ошибка) и возможностью закрытия |
| `Avatar` | Отображает аватар пользователя с возможностью показа статуса и инициалов, с поддержкой различных размеров и статусов (онлайн, оффлайн, печатает и т.д.) |
| `Badge` | Небольшой элемент для отображения статуса или количества с различными вариантами и размерами |
| `Button` | Универсальная кнопка с различными стилями (primary, secondary, success, warning, danger, ghost, link), размерами и состояниями (загрузка, отключена) |
| `CardSkeleton` | Заглушка для карточки, используемая во время загрузки данных, с анимацией для улучшения восприятия |
| `ChatStatusIndicator` | Индикатор статуса в чате (онлайн, офлайн, печатает), показывающий активность собеседника |
| `Checkbox` | Стандартный чекбокс с меткой, описанием и поддержкой ошибок для выбора одного или нескольких вариантов |
| `Chip` | Компактный элемент для отображения небольших фрагментов информации (например, тегов) с возможностью закрытия |
| `CodeBlock` | Блок для отображения форматированного кода с подсветкой синтаксиса, с поддержкой различных языков программирования |
| `Divider` | Горизонтальный или вертикальный разделитель с возможностью добавления текстовой метки |
| `ExampleAtom` | Пример простого атомарного компонента для демонстрации структуры |
| `Icon` | Отображает иконку из набора с различными размерами и настраиваемыми свойствами |
| `InlineCode` | Отображает небольшой фрагмент кода внутри текста с соответствующим стилем |
| `Input` | Поле для ввода текста, чисел и других данных с меткой, поддержкой ошибок и различными типами ввода |
| `Kbd` | Компонент для отображения клавиш клавиатуры в пользовательском интерфейсе |
| `Label` | Текстовая метка для элементов формы, обеспечивающая доступность и понятность интерфейса |
| `Link` | Ссылка с различными вариантами стиля (primary, secondary, success, warning, danger, ghost, underline) и размерами |
| `MessageStatus` | Индикатор статуса сообщения (отправлено, доставлено, прочитано) в виде чека |
| `MessageTimestamp` | Отображает временную метку сообщения в относительном или абсолютном формате (например, "10 минут назад") |
| `ProgressBar` | Индикатор выполнения задачи с заполнением в процентах |
| `RangeSlider` | Ползунок для выбора значения в заданном диапазоне с визуальной индикацией текущего значения |
| `Select` | Выпадающий список для выбора одного из нескольких вариантов с поддержкой ошибок и метки |
| `Skeleton` | Заглушка с анимацией, используемая во время загрузки данных для улучшения восприятия |
| `Spinner` | Индикатор загрузки в виде вращающегося элемента с различными размерами и вариантами |
| `StatusIndicator` | Индикатор статуса с иконкой и текстом (default, success, warning, error, info, primary, secondary) |
| `Switch` | Переключатель для включения/выключения опции в виде слайдера |
| `TableSkeleton` | Заглушка для таблицы, используемая во время загрузки данных |
| `Tag` | Тег для маркировки или категоризации с различными вариантами (нейтральный, первичный, успех, предупреждение, ошибка) |
| `Textarea` | Многострочное поле для ввода текста с меткой, поддержкой подсчета символов и ошибок |
| `TestButton` | Тестовый компонент кнопки для демонстрации функциональности |
| `Toggle` | Переключатель для включения/выключения состояния в виде слайдера |

Молекулы - это комбинации атомов, которые работают вместе как единое целое.

|----------|----------|
| `AttachmentPreview` | Компонент для предварительного просмотра прикрепленных файлов |

| `BurgerMenu` | Кнопка-гамбургер для открытия и закрытия меню с возможностью настройки размера и цвета |
| `ButtonGroup` | Группа связанных кнопок, отображаемых вместе для согласованного пользовательского интерфейса |
| `CanvasToolbar` | Панель инструментов для холста, содержащая инструменты для редактирования и взаимодействия с содержимым |
| `ChatHeader` | Заголовок чата, отображающий информацию о собеседнике и статусе |
| `CodeWithCopy` | Компонент для отображения кода с возможностью копирования в буфер обмена |
| `ConfirmDialog` | Диалог подтверждения для подтверждения важных действий пользователя |
| `CopyButton` | Кнопка для копирования текста в буфер обмена с визуальной обратной связью |

| `FieldHighlighter` | Инструмент для выделения полей формы, помогающий пользователю сосредоточиться на нужных элементах |

| `GraphNode` | Узел графа для визуализации данных |

| `GraphvizCluster` | Кластер для группировки узлов в Graphviz-визуализациях |
| `GraphvizClusterWithTitle` | Кластер с заголовком для группировки узлов в Graphviz-визуализациях |

## Организмы

Организмы - это сложные компоненты, состоящие из групп атомов, молекул и других организмов.

### Список организмов

| Компонент | Описание |
|----------|----------|
| `Accordion` | Интерактивный компонент, позволяющий сворачивать и разворачивать секции контента |
| `AccordionHeader` | Заголовок секции аккордеона, используемый для переключения видимости содержимого |
| `AccordionItem` | Отдельный элемент аккордеона, содержащий заголовок и панель |
| `AccordionPanel` | Панель аккордеона, в которой содержится контент, который можно свернуть или развернуть |
| `Breadcrumbs` | Навигационный компонент, показывающий путь пользователя от корневого уровня до текущей страницы |
| `ColumnManager` | Инструмент для управления видимостью и порядком колонок в таблице |
| `ConfirmDialog` | Диалог подтверждения для выполнения важных действий с подтверждением пользователя |
| `DataTable` | Интерактивная таблица с возможностью сортировки, фильтрации и пагинации данных |
| `ExpandableTableRow` | Расширяемая строка таблицы, позволяющая отображать дополнительную информацию при необходимости |
| `FolderTree` | Структура папок с возможностью разворачивания и сворачивания вложенных элементов |
| `Modal` | Модальное окно для отображения контента поверх основного интерфейса с возможностью закрытия |
| `Pagination` | Компонент для навигации между страницами данных |
| `Tab` | Вкладка в системе вкладок, содержащая заголовок и связанное с ней содержимое |
| `TabList` | Список вкладок, организующий визуальное представление доступных вкладок |
| `TabPanel` | Панель содержимого, связанная с определенной вкладкой |
| `TabPanels` | Контейнер для панелей вкладок, управляющий отображением активной панели |
| `Tabs` | Компонент системы вкладок для организации контента в отдельные секции |
| `Table` | Базовая таблица с возможностью настройки стилей и структуры |
| `TableBody` | Тело таблицы, содержащее основные строки данных |
| `TableCell` | Ячейка таблицы, используемая для отображения отдельных значений |
| `TableHeader` | Заголовок таблицы, содержащий названия колонок |
| `TableRow` | Строка таблицы, содержащая одну или несколько ячеек данных |
